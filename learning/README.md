# Learning Resources Directory

This directory contains explanations and demos for technologies used in the AgenticHarness project that you want to learn more about.

## Directory Structure

- `clickhouse/` - ClickHouse database tutorials and examples
- `kafka/` - Kafka messaging system guides  
- `redis/` - Redis usage patterns (including Vector Sets)
- `netty/` - Netty networking framework examples
- `spring-cloud/` - Spring Cloud microservices patterns
- `agentscope/` - AgentScope multi-agent framework guides

## Purpose

Each subdirectory contains:
1. **Basic Concepts** - Simple explanations of core concepts
2. **Why We Use It** - Specific reasons for choosing this technology in our project
3. **Hands-on Demos** - Practical code examples you can run locally
4. **Common Pitfalls** - Things to watch out for based on real experience

Use this as your personal learning reference while implementing the AgenticHarness system.
